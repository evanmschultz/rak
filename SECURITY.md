# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in rak, **please do not file a public issue**. Instead:

- Use GitHub's private vulnerability reporting: <https://github.com/evanmschultz/rak/security/advisories/new>
- Or email: <evan@hylla.io>

We'll respond within a few days and coordinate disclosure once a fix is available.

## Supported versions

Only the latest tagged release receives security fixes.

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |
